# AudioRecorderApp

Audio recording app built using SwiftUI and MVVM architecture following the MVVM design pattern. The app supports real-time audio recording with animated waveform visualization, playback controls, local file management, and background recording using AVFoundation.

## Features

- Audio recording using microphone
- Live animated waveform visualization
- Pause and resume recording
- Save recordings locally
- Recordings list with metadata
- Rename recordings
- Delete recordings
- Duplicate filename validation
- Audio playback support
- Play / Pause controls
- Seek audio using slider
- Display recording duration
- Display recording date & time
- Bottom sheet style audio player
- Background recording support

## Architecture

- SwiftUI
- MVVM Architecture
- AVFoundation
- Combine
- XCTest

## Unit Testing

Basic XCTest cases included for:

- Rename functionality
- Delete functionality
- Play/Pause state validation
